## Getting Started

### Prerequisites

- Node.js 16.x or higher
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/color-pick.git
cd color-pick
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash

```

4. Build for production:
```bash
npm run build
```
